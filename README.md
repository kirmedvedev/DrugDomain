# DrugDomain database
DrugDomain database (http://prodata.swmed.edu/DrugDomain/) reports [ECOD](http://prodata.swmed.edu/ecod/index_af2_pdb.php) domains of proteins that are targets for small molecules and drugs from [DrugBank](https://go.drugbank.com/)
